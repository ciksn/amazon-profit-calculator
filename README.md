# MarginGo 亚马逊多国利润率计算工具

第一版已包含：

- 多品类数据持久保存；每个品类独立选择国家，默认澳洲并优先展示。
- 成本、尺寸、重量跨国家共用；支持厘米/英尺、KG/LB。
- 8 个启用站点的当地售价、父品类、佣金与利润计算；法国、意大利、墨西哥暂时停用。
- 父品类关键词自动匹配佣金，也可手动覆盖。
- 每个国家使用独立商品尺寸分段，计算结果会显示该站点匹配到的尺寸段。
- 税额、产品成本、佣金、FBA、头程和单件利润逐项拆解。
- 管理员维护税率与汇率、货代价格、FBA 费阶、品类佣金。
- 本地 SQLite 保存，无需安装数据库或第三方依赖。

## 启动

电脑需要 Node.js 22 或更高版本。在本目录运行：

```powershell
npm.cmd start
```

## 当前计算口径

```text
扣除 VAT 后售价 = 含税售价 - 含税售价中的 VAT
单件利润 = 扣除 VAT 后售价 - 税费预估 - 平台佣金 - FBA - 头程 - 产品成本
利润率 = 单件利润 / 含税售价
```

产品成本和头程先以人民币计算，再按后台汇率换算为当地货币。平台佣金按含税售价计提，并支持当地售价分档。按 KG 报价的头程取实际重和体积重的较大值，所有站点的头程体积重除数固定为 6000；按方报价的头程直接用产品立方米数计算。FBA 体积重除数与头程完全独立，按各站点规则维护。

每个站点的测算表单均可直接修改当前货代单价；按 KG 的站点显示元/KG，按方的站点显示元/立方米，保存后立即用于该站点全部测算。

当前头程预估价：澳洲 4.8 RMB/KG、美国 7 RMB/KG、加拿大 8 RMB/KG、德国 12 RMB/KG；沙特 1500 RMB/m³、阿联酋 900 RMB/m³、日本 850 RMB/m³、英国 700 RMB/m³。

当前运营预估税费口径：

- 美国、澳洲、加拿大：税费和 VAT 均按 0。
- 沙特：税费为售价的 15%。
- 阿联酋：税费为售价的 5%。
- 日本：申报比例默认 15%，消费税默认 10%。申报价留空时按“售价×申报比例”计算，也可由运营直接覆盖；支持输入国内 10 位 HS 编码，按国际通用前 6 位匹配日本海关最新英文税则，再由运营确认日本 9 位细分项，并查询中国原产商品的 WTO/MFN 或 RCEP 税率。查询失败时仍可手动填写。税费公式为“申报价×关税比例＋（申报价＋关税）×消费税比例”。
- 英国、德国：进口/清关税费为产品成本的 10%，并另外从含税售价中拆分 20% / 19% VAT。
- 法国、意大利、墨西哥：当前不参与计算，也不出现在站点选择中。
- 佣金优先匹配具体品类；没有现成品类规则时自动使用该站点“其他类别”费率，不再沿用上一次匹配结果。日本的其他类别仍按 750 日元售价边界分档。

## GitHub Pages 版本

`docs/` 是可以直接部署的纯静态版本，不依赖 Node 或 SQLite。FBA、佣金、尺寸和国家规则位于 `docs/data/rules.json`；运营创建的品类和规则修改保存在当前浏览器，并可通过页面左侧的“导出备份 / 导入数据”迁移。

```powershell
npm.cmd run build:pages
python -m http.server 4174 --directory docs
```

`.github/workflows/deploy-pages.yml` 会在推送到 `main`、手动运行及每日定时任务中重新抓取日本海关税则并部署 Pages。当前仓库同时保留 `public/`、`server.js`、`lib/` 和 SQLite 版，便于本地管理或后续恢复服务端模式。日本海关英文税则仅供测算参考，最终税率以报关认定为准。

## 数据准确性说明

- FBA 已导入 marginGO 中澳洲、美国、沙特、阿联酋、加拿大、日本、英国、德国的截图/表格数据。
- 澳洲 FBA 使用资料注明的体积重除数 4000；货代头程仍按默认除数 6000。
- 澳洲 FBA 严格区分售价档和截图列出的特殊品类：低于 13 澳元、13–20 澳元、20 澳元及以上分别匹配对应费率。
- 美国 FBA 使用 2026-01-15 至 2026-10-14 费率，区分售价档、服装/非服装、尺寸段和发货重量；已加入 3.5% 附加费，续重按 4oz/1lb 整档向上取整，150lb 以上按实重计费。
- 沙特 FBA 以 25 SAR 为售价分界；沙特佣金表支持售价分档、阶梯佣金和最低 1 SAR 佣金规则（Grocery 除外）。
- 日本已同时导入售价低于 1000 日元的优惠 FBA 费率和常规费率，并按 1000 日元边界匹配。
- 加拿大已补齐 100g 标准件档、特殊大件、低价优惠、100g/500g 续重取整和 3.5% 附加费。
- 英国、德国已补齐全部大件/特殊大件费率，FBA 体积重除数为 5000，并支持指定品类按实重计费。
- 截图上传会在浏览器支持系统 OCR 时尝试识别；不支持时保留文件名并提示粘贴父品类文本。稳定生产版建议接入企业 OCR 或视觉模型。
